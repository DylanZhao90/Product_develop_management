### v3.5.0 — UI 品质升级：页面过渡动画 + 骨架屏 + 空状态 + 弹窗动效 + CSS去重 (2026-06-15)

#### ✨ 新增
- **页面过渡动画**：页面切换时 fadeIn + translateY 平滑过渡（0.35s cubic-bezier），替代生硬闪现
- **顶部加载进度条**：导航时页面顶部显示渐变进度条指示器，模拟 SPA 加载体验
- **骨架屏（Skeleton）**：项目/设计页面在数据渲染前显示 shimmer 加载占位动画
- **空状态提示**：产品搜索/筛选无结果时显示「📭 没有匹配的产品」友好提示
- **弹窗入场动画**：Modal 打开时弹性缩放 + 上移入场动画（cubic-bezier 弹簧效果）
- **设置头像动态显示**：个人设置页面头像自动取当前用户姓名首字，不再硬编码「张」
- **系统管理侧边栏徽章**：待审核注册用户数实时显示在侧边栏「系统管理」按钮上

#### 🔧 修复
- **重复 CSS 规则**：移除 `.toast` 重复定义（line 170-171）、`.page-header` 重复定义（line 74-78），减小文件体积
- **产品筛选空结果**：搜索/类型/状态筛选无匹配时不再显示空表格，改为友好空状态提示
- **设置头像 DOM 引用**：改用 `id="settingsAvatar"` 替代脆弱的选择器链 `.card:first-child div[style*="border-radius:16px"]`

#### 🔧 技术改进
- CSS 新增：`@keyframes shimmer`、`@keyframes modalIn`、`.empty-state`、`.page-progress`、`.skeleton` 系列

- `navigateTo()` 重构：新增进度条控制、骨架屏显示逻辑、`else if` 链替代多个 `if` 语句

### v3.4.0 — 审批流后台配置 + 项目审批节点动态同步 (2026-06-15)

#### ✨ 新增
- **审批流后台配置**：系统管理 → 审批流配置，支持新增/编辑/删除审批节点（节点名称 + 审批角色）
- **项目审批节点动态同步**：新建项目自动携带所有审批节点；后台修改审批流后，现有项目自动同步新增节点（显示为"待审批"）
- **审批流排序**：节点按 `order` 字段排序，确保显示顺序与后台配置一致

#### 🔧 修复
- **Node ID 冲突**：删除节点后新增节点不再产生 ID 冲突，改用时间戳+随机数生成唯一 ID
- **审批流与项目数据一致**：删除审批节点时自动清理所有项目中对应的审批记录
- **审批操作索引**：批准/拒绝操作使用排序后的审批流索引，消除增删节点后的下标错位

#### 🔧 技术改进
- 新增 `syncProjectsWithWorkflow()` — 每次审批流变更后自动同步所有项目审批记录
- `saveWorkflowNodes()` 不再直接调用，通过各操作函数触发 `syncProjectsWithWorkflow()`
- 启动时 `loadProjects()` 自动调用 `syncProjectsWithWorkflow()` 确保数据一致

### 核心技术
| 技术 | 用途 |
|------|------|
| SortableJS | HTML版拖拽排序 |
| @dnd-kit | React版拖拽排序 |
| ECharts 5 | 数据可视化 |
| localStorage | 布局 + 主题 + 用户 持久化 |
| localtunnel | 公网隧道穿透 |

### 设计系统
- 毛玻璃（backdrop-filter: blur）
- 渐变强调色（accent-gradient）
- 三档字体色（primary / secondary / muted）
- shadow-drag 拖拽阴影
- card / card-hover 边框过渡
- animate-in 入场动画
